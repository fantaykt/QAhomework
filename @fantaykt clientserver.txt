2) Что ткое HTTP и HTTPS
http это протокол передачи данных между браузером и сервером, https тоже самое но с методами шифрования и проверкой безопасности через сертификаты

3) HTTP методы
get post head put delete connect options trace patch

4) HTTP статус коды сервера
5хх ошибки все с сервера, основные 500 - internal server error (внутренняя ошибка сервера), 502 bad gateway (ошибочный шюз), 503 servise unavailable(сервис недоступен), 504 bad gateway (ошибочный шлюз), 520 unkown error(неизвестная ошибка)

5) Что такое ядро браузера 
это совокупность парсера, рендера и обработчика.

6) Какие браузеры какиие ядра используют
хромиумы - Blink, Edgehtml - ie, Gecko - мозила

7) Что такое API
Application Programming Interface - инструменты и функции в виде интерфейса через которая одна программа взоимодействует с другой

8. Что такое ендпоинты
шлюз для соединения серверных процессов с внешним интерфейсом, т е адрес куда принимает сообщения.

9) URL (URI, URL, URN)
URL - Uniform Resource Locator (определитель местонахождения), URN - Unifrorm Resource Name(имя ресурса), URI - Uniform Resource Identifier (идентификатор ресурса)

10) Идемпотентные HTTP методы
это те методы которые которые при повторе запроса имеет один и тот же эффект не меняющие состояние сервера (get, head, delete)

11) Безопасные HTTP методы
безопасными считатются методы которые не меняют состояния сервера (get head options)

12) Иденфикация, Аутентификация, Авторизация
идентификация - процедура выявления уникального признака однозначно определяющий его в системе
аутентификация - проверка подлинности.
авторизация - предоставление прав

13) Что такое IP
интернет протокол

14) Что такое октаты в DNS
первые две октета ID уникальный идентификатор транзакции, в третий октет входят QR-идентификация ответа или запроса, opcode тип запроса, aa - авторитетный или нет, tc - флаг означающий что будет следующий пакет,rd - флаг отключения промежуточных ответов. в четвертом октете  ra - флаг что сервер поддерживает рекурсию, z - всегда 000, rcode - успешность запроса либо ошика с кодом ошибки, пятый шестой октет - qdcount количество записей в секции запросов, седьмой восьмой октеты ancount - количество записей в секции ответы. девятый десятый октет - nscount - количество записей в секции авторитета, 11-12 октеты - arcount - количество записей в дополнительной секции для записей

15) Что такое порт, сколько портов у Linux сервера
порт это числовой идентификатор программы или процесса, порт записывается в заголовке пакета на транспортном уровне

16) Уровни OSI
7 уровней: физический - канальный - сетевой - транспортный - сеансовый - представления - прикладной

17) Хедеры http запросов
это строки в хттп содержащие имя-значение через двоиточие. четыре основные группы - general headers, request headers, response headersm, entity headers.
